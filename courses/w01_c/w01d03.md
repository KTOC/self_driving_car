* & 항상 주소를 나타낸다.
* 네모칸 하나가 한 바이트

***주소는 변수가 할당된 메모리 공간의 시작 주소를 사용*** -> 별딱지4개

* `&` 주소연산자
* `&a` a의 주소

>ex 9-2 257p typing 해봐

```c

#include <stdio.h>

int main(void)
{
    int a; // 일반변수 선언
    int *pa; // 포인터 선언

    pa = &a; // 포인터에 a의 주소 대입
    *pa = 10; // 포인터로 변수 a에 10 대입

    pritnf("포인터로 a값 출력 : %d\n", *pa);
    pritnf("변수명으로 a값 출력 : %d\n", a); // 변수 a 값 출력

    return 0;
}
```

결과
```
포인터로 a값 출력 : 10
변수명으로 a값 출력 : 10
```
