### 객체
* 객체 내부로 접근하기 위해서 이 객체를 참조하는 변수를 통해서 접근해야 한다.
    ```python
    >>> spell_object.print_spell()
    Hello World
    ```
* 어트리뷰트(속성)과 메쏘드
  * 위에서 객체 내부의 데이터를 담은 변수 `spell_object` 가 객체
  * `spell = "Hello World"` 는 `Atribute` 속성
  * `print_spell()` 그 안에 있는 함수는 `메쏘드`
  * `.` 은 속성접근자
